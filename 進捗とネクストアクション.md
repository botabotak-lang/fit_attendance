## 現在の進捗

### ✅ MVP + 本番データ層

| 機能 | 状態 |
|:---|:---|
| 打刻画面 | 氏名選択 → 出勤 / 退勤 / 外出 / 戻り |
| 今日の一覧 | 本日の打刻一覧表示 |
| 月次集計 | 日付範囲指定、出勤日数・勤務時間 |
| Excel出力 | 集計シート + 明細シート |
| **データ保存** | **Supabase**（`fit_attendance_punches` / `fit_attendance_employees`）。クライアントは `/api/*` 経由 |
| 社員マスタ編集 | `/admin` 最下部 |
| 打刻の編集・削除 | `/admin` 明細から |
| `/admin` 保護 | 任意: `ADMIN_BASIC_AUTH`（Basic）または Vercel Deployment Protection |

### 社員マスタ（初期シード）

鈴木・大竹・森田・深田・石橋・豊島（`supabase/fit_attendance.sql`。**修理アプリの社員シードとは別**）

## ネクストアクション（納品後・フェーズ2）

1. **休憩時間** - ヒアリング結果を反映
2. **直接現場勤務** - スマホ打刻の運用確認
3. **面談追加要件**（時間帯別ボタン、休暇ブランク、Supabase Auth による厳密な社長のみ認証 等）
